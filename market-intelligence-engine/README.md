\# Market Intelligence Engine



A lightweight, automated web scraper built with Node.js to gather and structure market data.



\## Features

\- \*\*Automated Fetching\*\*: Uses `axios` for reliable HTTP requests.

\- \*\*HTML Parsing\*\*: Uses `cheerio` for precise data extraction.

\- \*\*Data Export\*\*: Automatically formats and saves scraped data to CSV using `json2csv`.



\## Setup

1\. Clone the repository.

2\. Run `npm install` to install dependencies (`axios`, `cheerio`, `json2csv`).

3\. Execute the script: `node src/scraper.js`



\## Output

Saves structured data into the `/data` folder as `output.csv`.
